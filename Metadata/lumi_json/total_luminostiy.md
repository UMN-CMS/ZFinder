# Total Luminosity

These values come from running lumiCalc2 on the various json files. See:
https://twiki.cern.ch/twiki/bin/viewauth/CMS/LumiCalc#LumiCalc2

## Example Command

`lumiCalc2.py overview -i Run2012D.json`
